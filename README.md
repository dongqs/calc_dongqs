# CalcDongqs

Simple calculator

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'calc_dongqs'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install calc_dongqs

## Usage

    require 'calc_dongqs'
    Expression.new('1 + 2').result
